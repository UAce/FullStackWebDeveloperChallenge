# Frontend Client

Built with Create React App and Ant Design

## Project Structure

```
.
├── public                  # Public files
├── src                     # Source files
│   ├── components          # React Components files
│   └── index.tsx           # Entrypoint
└── README.md
```

## Get Started

```
npm start
```
You can access the web app via [`localhost:3000`](http://localhost:3000)
